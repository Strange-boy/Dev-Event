# DevEvents

## Purpose

**DevEvents** is a dedicated platform designed to connect developers and facilitate knowledge sharing. It serves as a hub for discovering upcoming hackathons, tech conferences, and meetups, fostering a vibrant community where developers can grow and collaborate.

## Features

We have implemented the following core features to provide a seamless user experience:

- ** immersive 3D Experience**: A stunning, interactive 3D background powered by Three.js to engage visitors immediately.
- **Curated Event Listings**: A comprehensive list of developer-focused events, making it easy to find relevant opportunities.
- **Feature Showcase**: A dedicated section highlighting the key benefits of the platform.
- **Newsletter Integration**: A subscription module for users to stay updated with the latest news and event announcements.
- **Modern Dark UI**: A sleek, developer-centric dark theme designed for visual comfort and aesthetic appeal.
- **Responsive Design**: Fully optimized layouts for mobile, tablet, and desktop devices.

## Tech Stack

This project is built using a modern, high-performance technology stack:

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Graphics**:
  - [Three.js](https://threejs.org/)
  - [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
  - [Drei](https://github.com/pmndrs/drei)
- **Icons**: [Heroicons](https://heroicons.com/)
- **Analytics**: [PostHog](https://posthog.com/)
- **Fonts**: [Geist](https://vercel.com/font)
